# Telegram-Бот ассистент.

Доступные функции:

1) Таймер
2) Генерация паролей - трёх видов
3) Переводчик - На 87 языков

# Технологии

Бот написан на **_`Python`_**. С использованием таких библеотек как
- PySocks==1.7.1
- yandexfreetranslate
- python-telegram-bot[ext]
- logging
- random
- datetime
- sqlite3
